#### 主题是typecho系统仿微信朋友圈开发的，只是自己学习写的一个主题，有需要的下载使用就好了，开源免费。

目前使用vue3进行重构，采用单页方式，放弃SEO

注意：使用本主题需要搭配RestfulIcefox插件，如果你已有安装Restful插件，请备份后关闭该插件，启用本站提供的插件
